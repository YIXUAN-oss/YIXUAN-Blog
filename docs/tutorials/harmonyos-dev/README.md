## 🚀 快速开始

### 安装 DevEco Studio

1. 访问官网下载 DevEco Studio
2. 安装并配置开发环境
3. 创建第一个项目
4. 运行到模拟器或真机

### 第一个应用

```typescript
@Entry
@Component
struct HelloWorld {
  @State message: string = 'Hello HarmonyOS'

  build() {
    Row() {
      Column() {
        Text(this.message)
          .fontSize(50)
          .fontWeight(FontWeight.Bold)
      }
      .width('100%')
    }
    .height('100%')
  }
}
```
