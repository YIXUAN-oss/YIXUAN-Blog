# 持续集成

> 搭建CI/CD流程

## 🔄 CI/CD流程

### 构建流程

```yaml
# .gitlab-ci.yml
stages:
  - build
  - test
  - deploy

build:
  stage: build
  script:
    - npm install
    - hvigorw assembleHap
  artifacts:
    paths:
      - build/outputs/

test:
  stage: test
  script:
    - npm run test

deploy:
  stage: deploy
  script:
    - ./deploy.sh
  only:
    - master
```

## 🚀 自动化构建

### DevEco Studio命令行构建

```bash
# 构建HAP
hvigorw assembleHap

# 清理构建
hvigorw clean

# 运行测试
hvigorw test
```
