## 💡 为什么学习 JavaScript？

- **Web 必备**：前端开发的核心语言
- **全栈开发**：Node.js 让 JS 可以用于后端
- **应用广泛**：网页、小程序、桌面应用、移动应用
- **生态丰富**：npm 上有海量的开源库
- **持续发展**：ECMAScript 标准不断更新

## 🎯 技术栈

### 前端框架
- ⚛️ **React** - 构建用户界面
- 💚 **Vue.js** - 渐进式框架
- 🅰️ **Angular** - 完整的应用框架

### 后端开发
- 🟢 **Node.js** - JavaScript 运行时
- 🚂 **Express** - Web 应用框架
- 🦅 **NestJS** - 企业级框架

### 构建工具
- 📦 **Webpack** - 模块打包工具
- ⚡ **Vite** - 下一代构建工具
- 🎨 **Babel** - JavaScript 编译器
